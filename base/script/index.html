# preloadе’Ңprefetch

preload е’Ң prefetch еҸҜд»Ҙз”ЁжқҘеҠ иҪҪjsгҖҒcssе’ҢеӣҫзүҮгҖӮйҖҡиҝҮlinkж Үзӯҫзҡ„refеұһжҖ§и®ҫе®ҡпјҢasеұһжҖ§и®ҫе®ҡеҠ иҪҪзҡ„ж–Үд»¶зұ»еһӢпјҢжүҖд»ҘдёҚдјҡйҳ»еЎһonloadзҡ„жү§иЎҢгҖӮд»–д»¬еҠ иҪҪзҡ„ж–Үд»¶дёҚдјҡз«ӢеҚіжү§иЎҢпјҢеҸӘиҙҹиҙЈеҠ иҪҪж–Үд»¶еҲ°жң¬ең°зЈҒзӣҳжҲ–иҖ…зј“еӯҳдёӯпјҢеҫ…еҲ°йңҖиҰҒзҡ„ж—¶еҖҷпјҢзӣҙжҺҘд»Һзј“еӯҳдёӯеҸ–е°ұеҸҜд»ҘдәҶгҖӮ

preload

еҠ иҪҪpreloadдјҳе…Ҳзә§жҜ”prefetchй«ҳпјҢи®©жөҸи§ҲеҷЁжҸҗеүҚеҠ иҪҪпјҢдҪҶжҳҜйғҪдёҚз«ӢеҚіжү§иЎҢ

prefetch

еҠ иҪҪpreloadдјҳе…Ҳзә§жҜ”prefetchдҪҺпјҢи®©жөҸи§ҲеҷЁеңЁз©әй—Ізҡ„ж—¶еҖҷиҮӘеҠЁеҠ иҪҪпјҢз”ЁдәҺйў„жөӢеҠ иҪҪеҸҜиғҪдјҡз”ЁеҲ°зҡ„ж–Үд»¶пјҢеҠ иҪҪе®Ңд№ҹдёҚдјҡз«ӢеҚіжү§иЎҢ

дҪҝз”ЁеңәжҷҜ

  • preload дёҖиҲ¬з”ЁдәҺд»Јз ҒеҲҮеүІе®Ңз”ҹжҲҗзҡ„bundleж–Үд»¶еҒҡpreloadпјҢиЎЁжҳҺдјҳе…ҲеҠ иҪҪпјҢдҪҶжҳҜдёҚйҳ»еЎһжөҸи§ҲеҷЁзҡ„еҠ иҪҪ

  • prefetch дёҖиҲ¬з”ЁдәҺи·Ҝз”ұзҡ„жҮ’еҠ иҪҪ

жіЁж„Ҹ

жіЁж„Ҹpreload е’Ң prefetch ж··з”ЁпјҢеҰӮжһңж··з”Ёе№¶дёҚдјҡеӨҚз”Ёиө„жәҗпјҢиҖҢжҳҜдјҡйҮҚеӨҚеҠ иҪҪгҖӮ

# vueйЎ№зӣ®

  • йЎөйқўеҝ…йЎ»иҰҒеҠ иҪҪзҡ„ж–Үд»¶жҜ”еҰӮapp.jsе’Ңchunk-vendors.jsпјҢд»–дҪҝз”Ёзҡ„preloadпјҢиЎЁжҳҺзҡ„жҳҜдјҳе…Ҳзә§й«ҳпјҢз«ӢеҚіеҠ иҪҪгҖӮиҖҢhome.jsе’Ңabout.jsдҪҝз”Ёзҡ„prefetch,иЎЁжҳҺз©әй—Іж—¶иҜ·жұӮпјҢйңҖиҰҒз”Ёзҡ„ж—¶еҖҷжүҚд»Һзј“еӯҳдёӯжӢҝгҖӮ

# asyncе’Ңdefer

async е’Ң defer еҸҜд»Ҙз”ЁжқҘејӮжӯҘеҠ иҪҪjsпјҢдёҚдјҡйҳ»еЎһonloadзҡ„жү§иЎҢгҖӮ

async

async иЎЁжҳҺж–Үд»¶жҳҜејӮжӯҘзҡ„пјҢеҸӘиҰҒиҜ·жұӮеҠ иҪҪе®Ңе°ұз«ӢеҚіжү§иЎҢпјҢдёҚз®ЎйЎөйқўеҠ иҪҪжҲҗд»Җд№Ҳжғ…еҶөпјҢеӨҡдёӘasyncж–Үд»¶пјҢдёҚдёҖе®ҡи°Ғе…Ҳжү§иЎҢ

defer

defer д№ҹжҳҜејӮжӯҘзҡ„пјҢдҪҶжҳҜе®ғеҠ иҪҪе®ҢдёҚдјҡз«ӢеҚіжү§иЎҢпјҢд»–дјҡзӯүеҫ…htmlеҠ иҪҪе®ҢеңЁиҝҗиЎҢпјҢдҪҶжҳҜе®ғжҖ»жҳҜеңЁDOMContentLoadedдәӢд»¶еүҚжү§иЎҢпјҢеӨҡдёӘdeferж–Үд»¶пјҢйЎәеәҸжү§иЎҢ

дҪҝз”ЁеңәжҷҜ

  • async дёҚдҫқиө–domе…ғзҙ пјҢд№ҹдёҚдјҡиў«е…¶д»–и„ҡжң¬дҫқиө–

  • defer дёҖиҲ¬з”ЁдәҺpolyfill.jsпјҢд»Јз ҒиҜӯжі•й«ҳдә®зӯүж–Үд»¶
